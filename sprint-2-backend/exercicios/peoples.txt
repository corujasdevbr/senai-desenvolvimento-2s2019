Olá, a empresa Peoples solicitou que você criasse a API para listar todas os funcionários da empresa. A peoples optou que tivesse apenas nome e cpf para cada um deles.
Você deverá ser capaz de:

Banco de Dados:
Script 01 : criar o banco de dados M_Peoples/T_Peoples; Tabela de Funcionarios contendo nome e cpf;
Script 02 : inserir os funcionários Catarina e Tadeu com os cpfs, 12345678901 e 10987654321 respectivamente;
Script 03 : selecionar todos os registros;

Backend:
Criar um projeto chamado Senai.Peoples.WebApi;
Criar o domínio correspondente;
Criar o repositório correspondente que contenha as ações de: listar, buscar por id, deletar, atualizar e inserir;
Criar o controller contendo os seguintes verbos:
GET 		/api/funcionarios
POST 		/api/funcionarios
GET 		/api/funcionarios/{id}
PUT 		/api/funcionarios/{id}
DELETE 		/api/funcionarios/{id}

Postman:
Você deverá criar uma coleção no postman chamada: Senai.Peoples.Manha com os endpoints correspondentes ao item acima;
Logo, você terá 5 itens dentro da pasta Funcionários com os verbos correspondentes;

GitHub:
Você deverá criar um repositório (2s2019-t2-sprint-2-backend-peoples ou 2s2019-t2-sprint-2-backend) ou adicionar ao seu repositório de backend existente o projeto do backend, os scripts do banco de dados (01, 02 e 03) e além disso, exportar a coleção do postman e disponibilizar essa coleção dentro do repositório.
