O Aplicativo Shirt Store é uma plataforma onde são mostrado as Camisetas de várias marcas.
Você foi contratado por ela para desenvolver uma API que será utilizada pelo front-end que trabalha na empresa.
A Shirt Store disponibilizou o script do banco de dados que já foi feito por um DBA.

A plataforma terá as tabelas abaixo
Usuários
Marca
Camisetas
Tamanhos
Cores

Regras
Terão dois tipos de usuários: os cliente e as empresas.
Os clientes podem se cadastrar na plataforma livremente.
Um Empresa pode ser cadastrada livremente já criando um usuário no inicio
As empresas podem conter vários usuários

Somente os usuários da empresa podem cadastrar uma camiseta.
Ex. O usuário1 da empresa Adidas adiciona as camisetas da marca.

Uma camiseta é composta por marca, descrição, tamanho e cor.
Uma camiseta tem somente uma marca vinculada
Uma camiseta pode ter vários tamanhos e cores

Não pode ter uma empresa cadastrada sem usuários

Suas Tarefas
Criar um projeto chamado Senai.ShirtStore.WebApi
Disponibilizar um documentação dos endpoits do projeto usando Swagger
Disponibilizar o teste dos endpoints utilizando o Postman
Disponibilizar os endpoints para as ações listadas abaixo
	- Login
	- CRUD de Marca
	- CRUD de Camisetas
	- CRUD de usuários
	- Inserção de novos usuários  que serão cadastrados por um usuário da empresa
	- Deleção de usuários  que será feita por outro usuário da empresa
	
Regras adicionais
	Deve ter um acesso que lista todas as marcas disponiveis
	Deve ter um acesso que lista todos os tamanhos disponiveis 
	Deve ter um acesso para que as empresas vejam somente as suas camisetas
	
Desafios extras
	Fazer um acesso para que os clientes vejam as camisetas selecionando uma determinada marca
